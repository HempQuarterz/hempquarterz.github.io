## ğŸ“ Design Notes

### Sacred Minimalism
- Clean, uncluttered interfaces
- Generous whitespace (breathing room)
- Focus on content over decoration
- Reverent, modern aesthetic

### Visual Hierarchy
1. **Primary Actions** - Large, prominent (Begin Session)
2. **Secondary Actions** - Medium size (View Journal, Mapper)
3. **Tertiary Actions** - Small, subtle (Edit, Delete)

### Color Psychology
- **Deep Indigo/Blue** - Night sky, reverence, depth
- **Golden Light** - Divine presence, illumination
- **Turquoise/Emerald** - Spirit, breath, life
- **Celestial Blue/Silver** - Heaven, clarity, purity

### Typography Scale
- **h1** - 2.5rem (40px) - Page titles
- **h2** - 2rem (32px) - Section headings
- **h3** - 1.5rem (24px) - Card titles
- **body** - 1rem (16px) - Main text
- **small** - 0.875rem (14px) - Labels, metadata
